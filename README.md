# 2026 WBC 東京巨蛋賽事懶人包

## 部署至 Cloudflare Pages

專案名稱：**2026WBC_Tokyo-ai**

詳細部署步驟請參考 [DEPLOY.md](DEPLOY.md)。

## 本地運行

1. 安裝依賴：`npm install`
2. 執行：`npm run dev`

> **注意**：Gemini API Key 已移至後端，請在 Cloudflare Secrets 中設定。本地開發時可建立 `.dev.vars` 供 wrangler 使用。
